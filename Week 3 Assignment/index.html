CSS Box Model

Content

Content

The actual content of the box, where text and images appear

Padding

Clears an area around the content, inside the border

Border

A border that goes around the padding and content

Margin

Clears an area outside the border, separating elements

CSS Selectors

Element Selector

Selects all instances of an HTML element

p {
    color: blue;
}

Class Selector

Selects elements with a specific class attribute

.highlight {
    background-color: yellow;
}

ID Selector

Selects a single element with a specific ID

#header {
    font-size: 2rem;
}

Descendant Selector

Selects elements that are descendants of another element

article p {
    line-height: 1.6;
}
